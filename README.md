to run the code have 2 terminals opened in vscode at the bottom. 

one for backend (using npx nodemon server.js)
the other for front end (using the keyboard shortcut ctrl+shift+R) to clear cache to refresh local browser. 

--->>
[terminal 1]
cd backend
node server.js

[terminal 2]
cd frontend
npx http-server -p 8080

[open in browser]
http://localhost:8080

[or]
right click and open via liver server on http doc



download Node.js for the commands to work.
